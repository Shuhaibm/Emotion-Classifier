Emotion Classifier

About

Welcome to my Emotion Classifier REST API! It can be used to find the emotion of a string. It will classify the string as one of the following:

  • sadness
  • surprise
  • fear
  • anger
  • joy
  • love

It is a Stacking Ensemble Model with 90.55% accuracy:
  • Base Models: Na√Øve Bayes, Random Forest as well as Logistic Regression for the Base Models
  • Meta Model: Random Forest, data concatenated with base model predictions
